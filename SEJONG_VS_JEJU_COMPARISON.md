## 🛠️ 11. 분석 방법론

### 11.1 사용 도구

**멀티프로세싱 시스템**
```python
# PDF 추출
python3 multiprocess_pdf_extractor.py sejong.pdf sejong_pages 150
python3 multiprocess_pdf_extractor.py jeju.pdf jeju_pages 150
```

- 15개 워커 프로세스
- 평균 100 페이지/초

1. **OCR 미사용**:
   - 텍스트 추출 미완료

   - 각 도시 10~15 페이지 검토
   - 전체 통계는 추정값

3. **수기 숫자**:
   - 인간 검증 결과 (우측 테이블) 미추출
   - 정확한 일치율 계산 불가

### 11.3 향후 개선

1. **OCR 완료**:
   - Tesseract 한글 팩 설치
   - 전체 172+126 페이지 텍스트 추출

2. **상세 분석**:
   - 후보자별 정확한 득표 집계

   - 기계-인간 검증 일치율 계산

3. **시각화**:
   - 득표율 차트
   - 지역별 분포 지도
