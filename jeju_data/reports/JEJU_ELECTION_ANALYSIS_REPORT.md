## 🤖 5. 기계-인간 검증 시스템

### 이중 검증 프로세스

**단계 1: 기계 분류 (투표지분류기운영팀)**
- ② 분류된 투표지수: 기계가 정상 인식한 투표지
- ③ 재확인대상 투표지수: 기계가 불확실하다고 판단한 투표지

**단계 2: 인간 검증 (심사·검열팀)**
- (a) 확인결과 - 분류된: 기계 분류 결과를 인간이 재확인
- (b) 확인결과 - 재확인대상: 재확인 대상을 인간이 직접 분류

**검증 공식:**
```
최종 득표수 = (a) + (b)
검증 통과 조건: ② = (a) AND ③ = (b)
```

### 샘플 데이터 검증 예시

#### 페이지 1 (이륙1동) - 이재명 후보
- 기계 분류 ②: 3,571표
- 재확인 대상 ③: (미표시)
- 인간 확인 (a): 3,571표
- 인간 확인 (b): 60표
- **최종 득표**: 3,631표
- **검증 상태**: ② = (a) ✓ 일치

#### 페이지 10 (도두5동) - 이재명 후보
- 기계 분류 ②: 1,372표
- 재확인 대상 ③: 58표
- 인간 확인 (a): 1,398표
- 인간 확인 (b): (수기 작성)
- **최종 득표**: 1,398 + (b)표

## 📈 6. 투표 패턴 분석

### 투표 유형별 분류

제주시 172페이지는 다음과 같이 구성:

1. **관외사전투표** (페이지 1~약 10)
   - 제주시 외 지역에서 사전투표한 유권자
   - 높은 선거인수 (평균 4,000~5,000명)

2. **관내사전투표** (페이지 약 11~50)
   - 제주시 내에서 사전투표한 유권자
   - 동/읍/면별로 구분

3. **선거일 투표** (페이지 약 51~170)
   - 투표일 당일 투표한 유권자
   - 투표함 번호별로 세분화 (제1투~제30투 등)
   - 가장 많은 페이지 차지

4. **특수 투표** (마지막 페이지들)
   - 거소·선상투표
   - 잘못 투입된 투표지 등

### 지역 특성

**제주시 행정구역 특징:**
- **동(洞)**: 이륙동, 이도동, 도두동, 남동, 오등동 등
- **읍(邑)**: 애월읍, 구좌읍 등
- **면(面)**: 추가 확인 필요

**세종시와 차이점:**
- 제주시는 **읍·면 지역 포함** (세종시는 대부분 동)
- 제주시가 **지리적으로 더 넓음**

## ⚠️ 11. 주의사항

### 데이터 품질
1. **수기 작성 숫자**
   - 우측 테이블 (a), (b) 열은 종종 손글씨
   - OCR 정확도가 낮을 수 있음
   - 수동 검증 권장

2. **페이지별 차이**
   - 일부 페이지는 스캔 품질이 다를 수 있음
   - 흐릿하거나 기울어진 페이지 존재 가능

3. **번호 건너뛰기**
   - 후보 번호 3, 6, 7은 없음
   - 자동 처리 시 주의

### 검증 시스템
1. **100% 일치를 기대하지 말 것**
   - 재확인 과정에서 약 3% 차이 발생 정상
   - ②와 (a)가 다를 수 있음 (인간 재검토 결과 반영)

2. **최종 집계는 (a)+(b) 사용**
   - 좌측 기계 분류는 참고용
   - 우측 인간 검증이 공식 결과
