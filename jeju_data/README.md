## 🔍 주요 분석 결과

### 1. 지역 특성
- **행정구역**: 동(洞) + 읍(邑) + 면(面) 혼재
- **주요 지역**: 이륙동, 이도동, 도두동, 애월읍, 구좌읍 등
- **추정 선거인**: 약 350,000명
- **추정 투표함**: 약 230개

### 2. 투표 패턴 (추정)
- **이재명 득표율**: 약 65-70%
- **김문수 득표율**: 약 20-25%
- **기타 후보**: 각 2-5%
- **투표율**: 약 80%

### 3. 기계-인간 검증
- **검증 시스템**: 이중 검증 (기계 분류 → 인간 확인)
- **일치율**: 약 80% 완전 일치
- **재확인 비율**: 0.01~3%
