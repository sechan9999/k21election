## 📈 불일치 패턴 분석

### 핵심 패턴

**1. 기계 분류(②) vs 인간 확인(a)**: ✅ **거의 항상 일치**
- 모든 검토 페이지에서 ② = (a) 확인
- 기계가 "분류된 투표지수"로 판단한 것을 인간도 동의

**2. 기계 재확인(③) vs 인간 재확인(b)**: ✗ **거의 항상 불일치**
- 대부분의 페이지에서 ③ ≠ (b)
- 일반적으로 ③ > (b) (기계가 재확인 대상으로 더 많이 분류)

### 불일치 예시

#### Page 3 (관내사전) - 가장 큰 불일치
```
이재명 (C1):
  기계: ② = 19,900 / ③ = 0
  인간: (a) = 19,900 / (b) = 452  ← 452표 추가 재확인!

김문수 (C2):
  기계: ② = 7,679 / ③ = 0
  인간: (a) = 7,679 / (b) = 280  ← 280표 추가 재확인!

이준석 (C4):
  기계: ② = 4,520 / ③ = 0
  인간: (a) = 4,520 / (b) = 128  ← 128표 추가 재확인!
```

**해석**: 기계가 재확인이 필요 없다고 판단(③=0)했으나, 인간 검증 과정에서 총 1,182표 중 874표를 재확인 대상으로 재분류

#### Page 4 (1투표함) - 반대 방향 불일치
```
이재명 (C1):
  기계: ② = 1,862 / ③ = 123
  인간: (a) = 1,862 / (b) = 49   ← 123 vs 49 (기계가 더 많이 재확인 판단)

  인간: (a) = 624 / (b) = 14    ← 43 vs 14 (기계가 더 많이 재확인 판단)
```

**해석**: 기계가 123표를 재확인 대상으로 판단했으나, 인간 검증에서는 49표만 실제 재확인
