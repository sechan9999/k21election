# 전체 126페이지 자동 추출 계획

## 현재 상황
- ✅ 14개 샘플 페이지: 수동 추출 완료
- ❌ 나머지 112개 페이지: 미추출
- 📊 목표: 세종시 전체 투표 결과 집계

---

## 옵션 비교

### 옵션 1: 수동 입력 (권장) ⭐
**방법**:
1. 각 페이지 이미지를 열람
2. 우측 테이블 (a), (b) 값 육안 확인
3. `extract_voting_results.py`에 데이터 추가

**예상 시간**: 3-4시간
**정확도**: 100%

**장점**:
- ✅ 완벽한 정확도
- ✅ 손글씨 인식 문제 없음
- ✅ 검증 불필요

**단점**:
- ❌ 시간 소요
- ❌ 반복 작업

---

### 옵션 2: OCR 반자동화
**방법**:
1. Tesseract + 한글팩 설치
2. 표 영역 자동 감지
3. OCR로 숫자 추출
4. 수동 검증 및 수정

**예상 시간**: 2-3시간 (구축 1시간 + 검증 1-2시간)
**정확도**: 70-85% (손글씨 인식률)

**장점**:
- ✅ 부분 자동화
- ✅ 반복 사용 가능

**단점**:
- ❌ OCR 정확도 낮음 (손글씨)
- ❌ 검증 필수
- ❌ 환경 구축 필요

---

### 옵션 3: 좌측 테이블만 자동 추출 (우측은 수동)
**방법**:
1. 좌측 테이블 (②, ③): OCR로 추출 (인쇄체)
2. 우측 테이블 (a, b): 수동 입력 (손글씨)
3. 자동 + 수동 혼합

**예상 시간**: 2-3시간
**정확도**: 좌측 95%, 우측 100%

**장점**:
- ✅ 좌측 자동화 (인쇄체 인식률 높음)
- ✅ 우측 수동 (정확도 보장)

**단점**:
- ❌ 여전히 수동 작업 필요
- ❌ 복잡한 워크플로우
