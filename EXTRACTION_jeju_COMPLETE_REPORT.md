## 📋 기계 분류 vs 인간 검증 분석

### 데이터 구조
각 페이지는 다음 두 가지 데이터를 포함:

**좌측 테이블 - 기계 분류 (㉮ + ㉯)**:
- ㉮ 분류된 투표지: 기계가 명확히 판독한 투표
- ㉯ 재확인 대상: 기계가 불확실하다고 판단한 투표

**우측 테이블 - 인간 검증 (a + b)**:
- a: 인간이 확인한 분류된 투표
- b: 인간이 확인한 재확인 대상

**최종 공식 집계**: (a) + (b)

### 샘플 데이터 분석 (페이지 1 기준)

```
기계 분류:
  - 분류된 투표지 (㉮): 5,270표
  - 재확인 대상 (㉯): 145표
  - 소계: 5,415표

인간 검증:
  - 확인된 분류 (a): 5,270표
  - 확인된 재확인 (b): 145표
  - 최종 집계: 5,415표

일치율: 100% (기계와 인간 검증 총합 동일)
재확인 비율: 145/5,415 = 2.68%
```

### 통계적 인사이트
- 재확인 대상 비율: 평균 ~2-3%
- 대부분의 투표는 기계가 정확히 분류
- 인간 검증 단계에서 최종 확정
- 불일치 발생 시 인간 검증 우선

```python
# OCR 기능 추가 (라인 65-75 근처)
import pytesseract
from PIL import Image

def extract_text_with_ocr(page_num: int) -> str:
    page = self.doc[page_num]
    pix = page.get_pixmap()
    img = Image.frombytes("RGB", [pix.width, pix.height], pix.samples)
    text = pytesseract.image_to_string(img, lang='kor+eng')
    return text
```
