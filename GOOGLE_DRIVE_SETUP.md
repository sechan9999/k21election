# Google Drive 공유 설정 가이드

## ⚠️ 현재 문제
폴더가 "제한됨" 모드로 설정되어 자동 다운로드가 불가능합니다.

---

## ✅ 해결 방법 (선택하세요)

## 방법 1: 폴더 전체 공유 설정 변경 ⭐⭐⭐⭐⭐

### 단계별 가이드:

**1단계: 폴더 열기**
```
https://drive.google.com/drive/folders/1TrwtadcgYBKHECU3YVkl4ee7gk79ScyM
브라우저에서 위 링크 열기
```

**2단계: 공유 설정 변경**
```
1. 폴더명에서 우클릭 (또는 폴더 열고 상단 공유 아이콘 클릭)
2. "공유" 메뉴 선택
3. "일반 액세스" 섹션 찾기
```

**3단계: 권한 변경**
```
┌─────────────────────────────────────┐
│ 일반 액세스                          │
├─────────────────────────────────────┤
│ [제한됨] ▼  → [링크가 있는 모든 사용자] ▼ │
│                                     │
│ 권한: [뷰어] ▼                       │
└─────────────────────────────────────┘

변경:
❌ "제한됨"
      ↓
✅ "링크가 있는 모든 사용자"

권한: "뷰어" (다운로드만 가능)
```

**4단계: 저장**
```
"완료" 또는 "저장" 버튼 클릭
```

**5단계: 확인**
```
변경 후 다시 채팅에서:
"설정 변경 완료" 라고 알려주세요
→ 자동으로 전체 폴더 다운로드 시작!
```

---

## 방법 2: 개별 파일 공유 (빠른 테스트) ⭐⭐⭐⭐

한 개 파일만 먼저 테스트:

```
1. 폴더 안에서 파일 하나 선택 (예: 서울시.zip)
2. 파일 우클릭 → "공유"
3. "링크가 있는 모든 사용자" 선택
4. 링크 복사
5. 여기에 붙여넣기

형식: https://drive.google.com/file/d/FILE_ID/view
```

---

## 방법 3: 직접 업로드 (즉시 가능!) ⭐⭐⭐⭐⭐

가장 확실한 방법:

```
1. 📎 클립 아이콘 클릭
2. 파일 선택 (서울시.zip, 부산시.zip 등)
3. 업로드 (한 번에 하나씩)
4. 자동 분석 시작!
```

**장점:**
- 공유 설정 불필요
- 즉시 실행 가능
- 100MB 이하 파일 지원
- 한글 파일명 자동 인식

## ❓ FAQ

**Q: 파일이 너무 큰가요?**
- 100MB 이하: 직접 업로드 ✓
- 100MB 이상: 폴더 공유 설정 필요

**Q: 한글 파일명이 문제인가요?**
- 아니요! 자동 인식 지원:
  - 서울시.zip → seoul
  - 부산광역시.7z → busan
  - 제주특별자치도.pdf → jeju

**Q: 압축 파일도 가능한가요?**
- 네! ZIP, 7Z, RAR 모두 자동 해제

**Q: 여러 도시를 동시에?**
- 폴더 공유: 전체 자동 처리 ✓
- 개별 업로드: 하나씩 처리
